net start MongoDB

CQL

- 命令不去分大小写.属性,标签关系类型,变量区分大小写
- create
- MATCH查询
- RETURN
- WHERE设置筛选条件
- delete删除节点/关系
- remove删除节点或关系的属性
- order by排序检索数据
- set添加或更新标签
