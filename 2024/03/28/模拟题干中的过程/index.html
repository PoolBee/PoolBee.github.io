辗转相除法

y == 0 ? x : gcd(y, x % y);

3 % 2 的计算是3%2的余数，结果为1。

2 % 3` 的计算是 2 除以 3 的余数，结果为 2。

辗转相除法思路：不比较大小的情况下，

如果是大除小，那么得到余数0或者其他。如果是0则返回除数就是最大公约数；如果是其他，则用小的数字除余数直到为0。为0的除数就是最大公约数。

如果是小除大，得到的余数是小的数，要交换位置，再大除小……

代码思路：x和y输入，如果不比较大小

为0则返回除数，其他则最终也是要返回除数，

递归解释：y == 0 ? x : gcd(y, x % y);

输入x,y等于0则返回x的值，

如果不等于0则递归调用gcd(y, x % y)

【不等于0则y为第一个参数[也就是x和y交换了位置]，然后x % y得余数作为第二个参数】

再次传入

2和3

gcd(y,x%y);

gcd(2,3)余数是2，不为0，则x变成y

gcd(3,2)余数是1，不为0，继续递归调用，x就变成y了

gcd(2,1)，因为3对2的余数为1，1不是0，继续递归调用，x和y交换

gcd(1,0)，2%1为0，则返回x，为1
