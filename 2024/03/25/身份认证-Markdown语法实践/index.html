生物认证的优点

  • 普遍性
  • 方便性
  • 不可复制性
  • 高安全性
  • 生物认证技术能够提供主动监控技术
    https://www.poolbee.top

指纹

  • 指纹认证书目前最为成熟的生物认证技术，在古代就被用来代替签字，证明身份。
  • 指纹认证具有方便、可靠、非侵害和成本低的等特点。

    指纹应用：公司单位的考勤指纹机、银行、证券等金融系统的门禁系统，笔记本电脑的指纹识别器，以及犯罪现场的指纹识别等。
    问题

    指纹模糊，不清晰，机器识别不了。

原理

指纹锁人的手指末端正面皮肤上凹凸不平产生的纹线。纹线有规律的排列形成不同的纹型。纹线的起点、终点、结合点和分叉点、称为指纹的细节特征点。

  • 指纹认证：就是通过计算机模式识别不同指纹的细节特征点。

掌纹

  • 掌纹是指手腕和手指之间的手掌表面上的各种纹线。掌纹与指纹一样，具有高度的唯一性。
  • 掌纹识别具有采样简单、图像信息丰富、用户接受程度高、不易伪造、受噪声干扰小等特点收到国内外研究人员的广泛关注。
  • 由于掌纹识别技术起步较晚，目前尚处于学习和借鉴其他生物特征识别技术的阶段。

手形

  • 经过生物学家大量验证，人的手型在一段时间具有稳定性，可以进行识别。

静脉

  • 静脉识别通过红外线摄像头获取手指、手掌或手背静脉的图像提取数字特征。比对时采用复杂的匹配算法对个人进行身份鉴定，确认身份。

虹膜

  • 虹膜是由相当复杂的纤维组织构成的，在人的一生中极其稳定，不容易更改。
  • 虹膜识别技术是将虹膜的可视特征转换成一个512字节的虹膜代码，这个模板被存储下来以便后期识别使用。
  • 虹膜识别被认为是最佳的生物认证技术具有准确性高、速度快、防伪性强的特点。

DNA
